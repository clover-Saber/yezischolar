# 椰子学术

一个 Zotero 插件，用于将文库中的文章同步到椰子学术，并生成文献综述。

https://github.com/clover-Saber/yezischolar.git

## 安装

1. 获取插件的 `.xpi` 安装包（Release 中下载）
2. 在 Zotero 中：**工具** → **插件** → **从文件安装…**
3. 选择 `.xpi` 文件并确认安装
4. 重启 Zotero

## 使用方法

### 1. 输入邀请码

1. 打开 Zotero → **编辑** → **首选项**
2. 在左侧找到 **"椰子学术"**
3. 点击 **"邀请码"** 进行填写

### 2. 同步文章

**方法 1：同步选中条目**

- 在 Zotero 中选择一个或多个条目
- 右键点击 → **"同步选中文章到椰子学术"**

**方法 2：同步所有条目**

- 点击 **工具** → **"同步所有文章到本地文件夹"**

### 3. 跳转网站

- 在 Zotero 中点击**工具**
- 右键点击 → **"打开椰子学术前端"**（如已填写邀请码，网页端自动确认）

或直接访问 https://yeziweb-production.up.railway.app/ （右上角填写邀请码）
